# Fellowships.io 

A clean little directory of **fellowships, grants, residencies, and VCs** for ambitious students and builders.

## Run

```bash
pip install -r requirements.txt
python db_utils/init_db.py   # build the table
python db_utils/seed.py      # fill it up
python app.py              
```

## Sitemap

| Route | What you get |
|---------------|------------------------|
| `/`           | Fellowships, accelerators & VCs |
| `/grants`     | Grants |
| `/residencies`| Residencies |
